# Smart-Fish-Farm-System

## Project Description
This project is a C++ based system designed to monitor conditions in a fish farm. 
The system helps farmers maintain a healthy environment for fish by tracking 
important water parameters.

## Objectives
- Monitor fish farm environmental conditions
- Alert the farmer when conditions become dangerous
- Demonstrate C++ programming techniques in a practical system

## Features
- Water temperature monitoring
- Water level monitoring
- Sensor data processing
- Warning system for harmful conditions

## Technologies Used
- C++
- Code::Blocks IDE
- IoT concepts
- Sensors simulation

## Program Structure
The project consists of the following components:

- main.cpp – Main program execution
- Sensor functions – Simulate water condition monitoring
- Data structures – Stack or Queue for system data handling

## How to Run the Program
1. Download the project files.
2. Open using Code::Blocks.
3. Compile the program.
4. Run main.cpp.
